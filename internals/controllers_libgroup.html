Libgroup main controller

class spam.controllers.libgroup.main.Controller(*args, **kwargs)¶

REST controller for managing library groups.

In addition to the standard REST verbs this controller defines the following REST-like methods:

  • subgroups (get_subgroups())

get_all(proj)¶

Return a tab page with a list of libgroups for a project and a button to add new libgroups.

This page is used as the library tab in the project view: spam.controllers.project.main.get_one().
